
------------------------------------------------------
RPCでNullReferenceExceptionが出る
------------------------------------------------------
RPCを書いたスクリプトをアタッチしているGOにPhotonViewがアタッチされていない
アタッチしてオブザーバーにRPCを書いたスクリプトを指定すること

------------------------------------------------------
AnimationEvent has no function name specified!
------------------------------------------------------
Animationウィンドウで五角形マークがあるのにイベントが設定されてないという意味。
イベントを設定するか、五角形マークを消す。

------------------------------------------------------
System.Dataがnamespaceにない的なの
------------------------------------------------------
System.Data.DLLをAssetフォルダ配下(シーンがある場所)に入れると直った

------------------------------------------------------
KeyNotFoundException: The given key was not present in the dictionary.
------------------------------------------------------
http://answers.unity3d.com/questions/216372/problem-with-mysql-connection.html
にあるmysql.dllをPluginフォルダ内のものと入れ替えると直った

------------------------------------------------------
未割り当てのローカル変数 'conn' が使用されました。
------------------------------------------------------
フィールド定義時に初期化されていない。初期化すること


------------------------------------------------------
・割り当て、呼び出し、インクリメント、デクリメント、待機、
　および新しいオブジェクトの式のみがステートメントとして使用できます。
------------------------------------------------------
最後の（）が無い
	誤　SelectData;
	正　SelectData();

------------------------------------------------------
・APIの参照が切れている
------------------------------------------------------
プロジェクトエクスプローラの参照を右クリックして参照の追加から追加する

------------------------------------------------------
・DLLを認識させたい
------------------------------------------------------
Assetフォルダ配下に入れれば自動で認識される
		参考：http://www.wisdomsoft.jp/662.html

------------------------------------------------------
・Toggle Groupを複数設置すると効かなくなる
------------------------------------------------------
①親パネルのWidthとHightが大きすぎてToggle Groupのボタンにカブってないか
　→親パネルのWidthとHightを小さくする

------------------------------------------------------
・Object reference not set to an instance of an objectエラー
------------------------------------------------------
①Resources.Loadで指定している文字列が間違っていないか
②Start()内などで行っているコンポーネント取得先の指定を間違っていないか
　GetComponentしているオブジェクトに本当にそのコンポはあるか確認する

------------------------------------------------------
・マウスオーバーとかのコールバックが取れない
------------------------------------------------------
①ネームスペース
　using UnityEngine.UI;
　using UnityEngine.EventSystems;
　はあるか確認
②IF（IPointerEnterHandlerとか）は継承しているか確認

------------------------------------------------------
・Static classes must derive from object
------------------------------------------------------
MonoBehaviourクラスを継承したクラスは staticにはできないという意味のエラー


------------------------------------------------------
・UnityEditor.UI.dll' is in timestamps but is not known in guidmapper...
------------------------------------------------------
uGUIがイカれたときはインストールフォルダのGUISystemを一旦移動したあと起動して再度戻して起動すると直る
