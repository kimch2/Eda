
------------------------------------------------------
・Toggle Groupを複数設置すると効かなくなる
------------------------------------------------------
A.
	①親パネルのWidthとHightが大きすぎてToggle Groupのボタンにカブってないか
	　→親パネルのWidthとHightを小さくする

------------------------------------------------------
・Object reference not set to an instance of an objectエラー
------------------------------------------------------
A.
	①Resources.Loadで指定している文字列が間違っていないか
	②Start()内などで行っているコンポーネント取得先の指定を間違っていないか
	　GetComponentしているオブジェクトに本当にそのコンポはあるか確認する

------------------------------------------------------
・マウスオーバーとかのコールバックが取れない
------------------------------------------------------
A.
	①ネームスペース
	　using UnityEngine.UI;
	　using UnityEngine.EventSystems;
	　はあるか確認

	②IF（IPointerEnterHandlerとか）は継承しているか確認

------------------------------------------------------
・Static classes must derive from object
------------------------------------------------------
A.
	MonoBehaviourクラスを継承したクラスは staticにはできないという意味のエラー


------------------------------------------------------
・UnityEditor.UI.dll' is in timestamps but is not known in guidmapper...
------------------------------------------------------
A.
	uGUIがイカれたときはインストールフォルダのGUISystemを一旦移動したあと起動して再度戻して起動すると直る
